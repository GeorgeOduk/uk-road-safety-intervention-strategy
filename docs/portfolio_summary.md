# Portfolio Summary

## Project

UK Road Safety Intervention Strategy

## One-line Summary

A consulting-style analytics project using SQL Server, Power BI, Excel and PowerPoint to identify where road-safety investment should be targeted to reduce killed or seriously injured casualties.

## Tools Used

- SQL Server
- Power BI
- Excel
- PowerPoint
- GitHub

## Business Question

Where should limited road-safety investment be targeted to reduce KSI casualties most effectively?

## Key Outputs

- SQL reporting layer
- Power BI dashboard
- Excel QA workbook
- Executive PowerPoint deck
- Recommendation-led GitHub documentation

## Dashboard Pages

1. Executive Summary
2. Who is Most at Risk?
3. Where and When is Risk Concentrated?
4. Intervention Targeting

## Executive Recommendation

Refocus road-safety investment on urban pedestrian hotspots, higher-speed severity corridors, high-rate authorities and equity-sensitive targeting.
