# Data Sources

## Department for Transport Road Safety Open Data

This project uses Department for Transport road safety open data covering collisions, vehicles and casualties.

Files used:
- Collisions - last 5 years
- Vehicles - last 5 years
- Casualties - last 5 years

The DfT data supports analysis of:
- casualty severity
- KSI casualties
- road-user groups
- local-authority concentration
- speed-limit context
- lighting conditions
- time and day patterns

## Office for National Statistics Population Estimates

ONS local-authority population estimates were used to calculate population-adjusted severe-harm metrics.

The key enriched metric is:

- KSI per 100,000 population

This helps compare local authorities more fairly by adjusting for population size.

## Analytical Limitation

This project is a strategic portfolio analysis. It does not prove causation. The recommendations should be interpreted as pilot prioritisation guidance and would require local validation before implementation.
